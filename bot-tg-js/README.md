# 🤖 Telegram Bot Portfolio

Bot multifuncional com **clima**, **gerenciador de tarefas** e **admin panel**. 



## ✨ Comandos

| Comando | Função |
|---------|--------|
| `/start` | Menu interativo |
| `/ping` | Latência + uptime |
| `/clima <cidade>` | Previsão do tempo |
| `/tarefa add <texto>` | ➕ Nova tarefa |
| `/tarefas` | 📋 Lista tarefas |
| `/tarefa done/rm <id>` | ✅/🗑️ Gerenciar |

## 🚀 Quick Start

```bash
git clone https://github.com/SEUUSER/telegram-bot-portfolio
cd telegram-bot-portfolio
npm i
cp .env.example .env  # BOT_TOKEN + WEATHER_API_KEY
npm run dev
```

## 🌐 Deploy Grátis (Render)

1. [render.com](https://render.com) → **New Web Service**
2. **Build:** `npm i` | **Start:** `npm start`
3. Env vars: `BOT_TOKEN`, `WEATHER_API_KEY`, `ADMIN_USER_ID`
4. **Deploy** ✅ 24/7 grátis!
